Використовуючи ТС, створіть таку функцію, яка розраховує вартість морозива: 
Магазин пропонує 2 розміри морозива: 
- маленький стакан (10 грн.) 
- великий стакан (25 грн.). Морозиво доповнюється однією або декількома начинками (не менше однієї): 
    - шоколад (+5 грн.); 
    - карамель (+6 грн.); 
    - ягоди (+10 грн.). 
На морозиво можна додатково (за бажанням) посипати маршмеллоу (+5 грн.). Вхідні параметри вводяться користувачем через promt. 
Створіть об’єкт морозива 
const icecreame = { 
        'small': {'price': 10}, 
        'big': {
            'price': 25, 
            'topings': { 
                'chocolate': 5, 
                'caramel': 6, 
                'berry ': 10 
            }
        } 
        'marshmalow': {'ціна': 5} }; 
Застосуйте функцію buyicecream для розрахунку вартості та функцію chooseToppings для вибору та розрахунку начинок. Користувач може вибрати кілька доповнень. 
    Додатково. Користувач повинен вибрати одну з начинок. Якщо користувач не вибрав жодного доповнення, знову запропонуйте йому ввести назву доповнення: 'You must choose one of the toppings. Available: chocolate, caramel, berry.'
